### 2. **"Recently Viewed" - المنتجات المشاهدة مؤخراً** ⭐⭐⭐⭐

**التصميم:**

**التنفيذ:**
```tsx
// src/hooks/useRecentlyViewed.ts
export function useRecentlyViewed() {
  const [viewed, setViewed] = useState<Product[]>([]);

  const addToViewed = (product: Product) => {
    const recent = JSON.parse(
      localStorage.getItem('recently_viewed') || '[]'
    );

    // إضافة المنتج في البداية
    const updated = [
      product,
      ...recent.filter(p => p.id !== product.id)
    ].slice(0, 10); // حفظ آخر 10 منتجات

    localStorage.setItem('recently_viewed', JSON.stringify(updated));
    setViewed(updated);
  };

  return { viewed, addToViewed };
}
```

**الموقع:**
- أسفل صفحة المنتج
- الصفحة الرئيسية
- صفحة السلة

**الوقت المتوقع:** يوم واحد

```tsx
  const percentage = (stock / totalStock) * 100;

    <div className={cn("p-4 rounded-lg", statusColors[status])}>
      <div className="flex items-center justify-between">
```
